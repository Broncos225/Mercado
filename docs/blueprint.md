# **App Name**: Mercado

## Core Features:

- Shopping List Creation: Create a shopping list with items, quantities, and planned values.
- Purchase Selection: Mark items as purchased.
- Value Editing: Edit the actual value of purchased items.
- Planned vs. Actual Comparison: View a comparison of planned vs. actual expenses for market shopping.
- Shopping List Persistence: Persist all Shopping Lists to the Firestore database

## Style Guidelines:

- Primary color: A warm and inviting pastel green (#A7D1AB), reminiscent of fresh produce and growth, setting a positive tone for budget management.
- Background color: A desaturated, very light green (#F0F4EF) that complements the primary color and provides a calming backdrop, focusing user attention.
- Accent color: A soft coral (#F28482), drawing the user's eye to interactive elements like 'add' or confirmation buttons.
- Body and headline font: 'PT Sans', a humanist sans-serif to blend warmth with a modern aesthetic.
- Simple, outline-style icons to represent different shopping list categories.
- Clean, card-based layout for each shopping list item.
- Subtle animations when adding items to the list or marking them as purchased.
